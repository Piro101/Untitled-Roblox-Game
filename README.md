## Getting Started
To build the place from scratch, use:

First, ensure rojo, rokit, and wally-package-types are installed

```bash
rokit install
wally install
rojo sourcemap --output sourcemap.json --include-non-scripts
wally-package-types --sourcemap sourcemap.json Packages/ ServerPackages/
rojo serve
```
